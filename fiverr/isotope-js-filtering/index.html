LiVE CARDS

LiVE CARDS is a theortical product that I and three others were tasked to create as a part of the New Media Certification course. LiVE CARDS is a physical greeting card that utilizes an App Clip code to house 3D assets. Upon scanning the code your ios device will download a segment of an app to reveal exciting visuals through augmented reality. I created all the design work, cards, 3D visuals, and any other asthetic elements related to this project which I have showcased below.
